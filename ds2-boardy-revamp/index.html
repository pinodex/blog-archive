Hey there! It's been a long time since my last update. I almost forgotten the fact that this blog exists. Whatever.

Two months ago, I published the first version of Boardy, the forum software that I am working on. It was half good and half bad because of it having an incomplete features, and undocumented code.

So I decided to rewrite the software from scratch. Not totally from scratch as I will be using some old codes and the same framework. Some things might not be compatible with the new code, but I assume that no one used the current code so there's no worries in doing migrations.

That's it for now. Stay tuned for updates!
