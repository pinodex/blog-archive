Introduction

I just finished a weekend project, a simple PHP forum app called Boardy. This is the successor of the previous forum app I made before. This is also my first Silex project. I used Silex because I wanted to try different lightweight frameworks, and it was great!

What can Boardy do?

Pretty much all basic features a forum app must have. No plugins, media uploads and moderation tools tho.

There's nothing else need to talk about this right now. You can view the project at github.com/pinodex/boardy.
